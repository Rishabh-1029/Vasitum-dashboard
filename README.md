# React Vasitum Dashboard Project

This project is a *practice React dashboard* created as part of an *on-site internship training task* at *Vasitum Tech*.  

The goal was to develop a functional dashboard according to a specific *Figma design* provided by the organization. The project involved integrating various features, styling, and API communication as per the requirements.

---

## *Technologies Used*
- *HTML*
- *CSS*
- *JavaScript*
- *React.js*
- *REST API Integration*

---

## *Features*
- Fully responsive dashboard layout
- Data fetching from REST APIs
- Interactive UI components
- Styled according to the Figma design provided

---

## *Project Purpose*
This project was completed as part of my internship training at Vasitum Tech to gain practical experience in:
- Implementing front-end designs
- Working with React components
- REST API integration
- Translating Figma designs into functional applications
